# SillyTavern Admin 전체 기능 구조화 문서

## 1. 개요

### 1.1 목적
SillyTavern Admin의 모든 기능을 체계적으로 정리하고, 구현 상태를 추적하며, 최적의 메뉴 구조를 설계한다.

### 1.2 현재 상황
- **구현 완료**: 약 30-40%
- **구현된 기능**: 캐릭터 관리, 설정 관리 (일부)
- **구현 미완료**: 채팅 관리, 그룹 관리, 미디어 관리, 확장 기능 관리 등

### 1.3 문제점
- 기능의 복잡도와 메뉴 구조가 일관성이 없음
  - 캐릭터 관리: 간단한 기능인데 별도 메뉴
  - World Info: 복잡한 기능인데 설정 메뉴 안에 포함
- 메뉴 위계 구조가 명확하지 않음
- 구현 상태 추적이 어려움

---

## 2. 전체 기능 목록 및 분류

### 2.1 기능 분류 기준

#### A. 기능 유형별 분류
1. **관리 기능** (Management): 데이터 CRUD 작업
2. **설정 기능** (Configuration): 시스템/동작 설정
3. **유틸리티 기능** (Utility): 편의 기능, 도구

#### B. 복잡도별 분류
1. **단순** (Simple): 기본 CRUD만 있는 기능
2. **중간** (Medium): CRUD + 검색/필터링
3. **복잡** (Complex): CRUD + 고급 기능 + 설정

#### C. 사용 빈도별 분류
1. **높음** (High): 자주 사용하는 기능
2. **중간** (Medium): 가끔 사용하는 기능
3. **낮음** (Low): 가끔 사용하는 기능
